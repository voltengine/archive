# Archive

📁 Volt package repository.

## Getting Started

### Setup

📝 Create following `./.env` file:

```
PORT={port}
GITHUB_CLIENT_SECRET={oauth-app-secret}
```

⚙️ Modify `./src/config.js` as you fancy. Remember to set matching GitHub client ID.

💻 Install Node.js dependencies with:

```
npm install
```

### Running
```
npm test
```
